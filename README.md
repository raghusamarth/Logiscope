Cyber Monitoring Dashboard

Overview
This project visualizes system activity across:
- Firewall
- Load Balancer
- Web Server

It distinguishes between **Cyber and Non-Cyber events** and provides insights into system behavior.

Features
- Interactive dashboard (Streamlit)
- Firewall drill-down (IP + Port)
- Cyber vs Non-Cyber classification
- Multi-layer monitoring

How to Run
1. Install dependencies:pip install -r requirements.txt
2. Run Dashboard: streamlit run dash.py
